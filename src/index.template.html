About this Project

An open source and noncommercial side-scrolling space shooter browser game for your PC and Mac

Repo: GitHub

Technologies

JavaScript, HTML, CSS, WebGL (using PixiJS)

Cool parts of this Codebase

behavior.js - See how the NPCs make decisions based on their environment and state.

mainReducer.js - Ever since first trying Redux I've really loved the idea of reducers, named actions and pure functions for managing state. So the state reducer pattern was the first tool I reached for when building Sublight Patrol, and it ended up being a really good fit for a game like this where you have a lot of different entities and systems all needing to interact with the same state in complex ways.

scene003.js - You can think of the numbered scene files in the story folder as the "screenplay" for the different scenes in the game. Dig into scene003.js to see how "Mission 3: Capturing the Prototype" is put together, e.g. what NPCs are in play and what their behavior is, and how the different story beats are triggered as the player works her way through the objectives presented.
